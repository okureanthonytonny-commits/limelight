# limelight
  
  This is a project I vibe coded and it started to feel like things were going in the wrong way, so I decided to rebuild it in a separate repo as a v2, while basing on what I had learned in this one. Here's how it went.

## What limelight is

 This is supposed to be a super simple "thing". A single vendor marketplace with two reactjs frontends + fastapi + postgresql.  

## 1. Architecture

 I read and watched lots of guys talking about being the architect and let the AI handle the coding. I was super lazy doing this. So, I just chatted with deepseek about it, cameup with a project_architecture.json and helping generate detailed prompts for copilot. I was in a "let's just do it and see whether it works"

## 2. Project structure

 I asked deepseek to write the structure.sh, after a little refinement it gave me code in restructure.sh and things were looking quick.

## 3. The implementation

 So, I was doing almost nothing after that except copy-pasting code and prompts, simple adjustments, afterall these models "know how to code" and I assumed they can do it all so well.

## How it started to feel wrong

 After copy pasting several times and clicking keep, copying the errors and passing them on to deepseek, asking copilot to findout what the problem was, I really started getting nervous because the errors were slowly getting louder and louder. There came a point I took 30 minutes copy pasting hoping AI would fix the code it wrote and it was a clear sign that I had wasted my time, emotion, eye health!

## Here's why I think it got annoying

## 1. Context map

 I built very little context about the entire project, I knew the project structure and architecture were accessible to copilot, deepseek afterall this is a basic project. After copy pasting several times, I totally didn't know where I was going with the project, then I started to look through the code.

 I started to map out one thing at atime, reading one function at a time, and I was bored, and fraustrated already. When things started to click, I had spent lots of hours. F**** it all, I just created a new project where I'd identify the bugs before they build up.
